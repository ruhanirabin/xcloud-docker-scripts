# xcloud-docker-scripts
Scripts to use with xCloud Nginx+Docker stack to deploy OSS apps

## Current Scripts
Scripts are modified from original `docker-compose.yml` and `.env` files so it can be installed using xCloud Nginx+Docker stack

[Twenty CRM](twenty-crm)


## How to use
1. Download the appropriate yml and env files
2. Modify the necessary details in the yml OR env file
3. Paste the contents of the yml file into the xCloud Nginx+Docker stack
4. Deploy and follow any specific instructions for the app in their documentation (readme)

## How to contribute
1. Fork this repository
2. Clone the repository to your local machine
3. Create a new branch
4. Make your changes
5. Commit your changes
